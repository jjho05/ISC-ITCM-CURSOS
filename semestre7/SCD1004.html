Conmutación y Enrutamiento en Redes de Datos

Clave: SCD-1004 | Créditos SATCA: 2-3-5

Caracterización de la Asignatura

Esta asignatura aporta al perfil del Ingeniero en Sistemas Computacionales habilidades para diseñar, configurar y administrar redes, así como para desarrollar software que apoye la productividad. Desarrolla las capacidades para el diseño e implementación de soluciones en redes de datos LAN y WAN en base a las normas y estándares vigentes.

La importancia de esta asignatura radica en la necesidad de optimizar procesos empresariales mediante el uso de tecnologías de la información y redes de datos. Es subsecuente a la asignatura de Redes de Computadoras y desarrolla las competencias necesarias para cursar Administración de Redes.

Competencia General de la Asignatura

  • router Diseña, instala y configura redes LAN y WAN, tanto alámbricas como inalámbricas, aplicando normas y estándares vigentes para la solución de problemas de conectividad.

Temario y Competencias por Unidad

1. Direccionamiento IP y Enrutamiento

Competencia: Establece un diseño y configuración en redes de datos para satisfacer las necesidades de conectividad y seguridad, analizando la funcionalidad de algoritmos y protocolos de enrutamiento.

  • 1.1 Direccionamiento IP.
    • 1.1.1 Direccionamiento con clase (VLSM).
    • 1.1.2 Direccionamiento sin clase (CIDR).
  • 1.2 Enrutamiento estático y dinámico (vector-distancia, de enlace).
  • 1.3 Protocolos de enrutamiento (RIP, EIGRP, OSPF).

2. Conmutación de redes LAN

Competencia: Aplica las diferentes tecnologías y metodologías de conmutación para el diseño de una red conmutada.

  • 2.1 Segmentación de dominio de colisión y broadcast.
  • 2.2 Métodos de conmutación.
  • 2.3 Tecnologías de conmutación (VLAN, VTP, STP).

3. Tecnologías WAN

Competencia: Analiza y configura enlaces WAN para diseñar una red, utilizando diferentes tecnologías y topologías, y elabora un proyecto de red de datos.

  • 3.1 Introducción a redes WAN.
  • 3.2 Enlaces: dedicados y públicos.
  • 3.3 Topologías WAN.
  • 3.4 Tecnologías WAN: PPP, XDSL, frame relay, ISDN, ATM.
  • 3.5 Configuración de dispositivos WAN.
  • 3.6 Niveles de seguridad en redes VPN.

4. Tecnologías inalámbricas

Competencia: Integra tecnologías inalámbricas en un proyecto de redes WLAN/WAN y aplica mecanismos de seguridad para mejorar la fiabilidad.

  • 4.1 Clasificación: WPAN, WLAN, WMAN, WWAN.
  • 4.2 Estándares y protocolos: bluetooth, Infrarrojo, Wi-Fi, Wi-Max.
  • 4.3 Dispositivos y configuración.
  • 4.4 Seguridad: WEP, WAP, WPA-PSK, WEP2, filtrado de MAC's.

Software y Herramientas Propuestos

Para las prácticas se recomienda el uso de herramientas de simulación para el diseño y configuración de redes, así como software de análisis.

  • Simuladores de redes como Cisco Packet Tracer o GNS3.
  • Software de análisis de protocolos como Wireshark.
  • Equipos de red físicos (routers, switches) para prácticas de configuración real.
